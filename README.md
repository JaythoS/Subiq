# Subiq Support

Welcome to Subiq support.

Subiq is an iOS app designed to help users track subscriptions, renewal dates, free trials, reminders, and recurring spending in one place.

## Features

- Track subscription renewal dates
- Monitor free trials before they convert into paid plans
- Set one or multiple reminders in days or hours
- View weekly and monthly renewal timelines
- See spending insights across subscriptions
- Unlock unlimited subscriptions with Subiq Premium

## Need Help?

If you experience any issues while using Subiq, please review the following:

### Common Topics

- Subscription reminders and notification permissions
- Premium purchase and restore purchases flow
- Free trial tracking and renewal setup
- App icon and subscription matching behavior
- Currency conversion and spending insights

### Troubleshooting

Before contacting support, please try:

1. Make sure you are using the latest version of Subiq.
2. Check that notification permissions are enabled in iPhone Settings if reminders are not being delivered.
3. Reopen the app after restoring purchases or updating premium access.
4. Verify your subscription renewal date and reminder settings if a notification does not appear as expected.
